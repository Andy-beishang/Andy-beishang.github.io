Linux安装软件：

  1. 包管理工具
    • yum CentOS
      ~ yum search nginx
      ~ yum install nginx
      ~ yum erase nginx / yum remove nginx
      ~ yum info nginx
      ~ yum list installed | grep nginx
    • rpm Redhat
      ~ rpm -ivh 下载的rpm包文件名
      ~ rpm -e
      ~ rpm -qa
    • apt Ubuntu
  2. 源代码构建安装
    安装Redis官方最新版
    ~ wget http://download.redis.io/releases/redis-5.0.7.tar.gz
    ~ gunzip redis-5.0.7.tar.gz
    ~ tar -xvf redis-5.0.7.tar
    ~ cd redis-5.0.7
    ~ make && make install
    安装Git官方最新版
    ~ wget https://mirrors.edge.kernel.org/pub/software/scm/git/git-2.24.1.tar.xz
    ~ xz -d git-2.24.1.tar.xz
    ~ tar -xf git-2.24.1.tar
    ~ cd git-2.24.1
    ~ yum install -y curl libcurl-devel
    ~ ./configure –prefix=/usr/local
    ~ make && make install

redis的核心数据类型：
1.字符串
set key value
get key
strlen key
append key value2
mset k1 v1 k2 v2 k3 v3…
mget k1 k2 k3…
键值对的值为数字时：
incr key
incrby key 增加数字
decr key
decrby key 减少数字
取出字符串中某一部分
getrange key 下标范围
eg: getrange username 5 7(注意，下标都是闭区间)
setrange key 开始下标 替换的新的字符串

哈希（hash）
存储值为哈希表的键值对
hset key field(vk) vv
hmset key vk1 vv1 vk2 vv2 vk3 vv3…
hget key vk
hmget key vk1 vk2
hgetall key获得该键的所有字段和字段对应的值
hkeys key获得该键所有字段
hvals key获取该键的所有的字段对应的值

exists key 判断该key是否存在
hexists key field 判断该key是否存在某一个字段（field）
type key

3.列表（list）

lpush(leftpush) key value1 v2 v3 …
rpush(rightpush) key value1 v2 v3 …
lpop key
rpop key
lrange key start end(下标范围)
llen key
lindex key index 根据下标查元素
linsert ？自己查用法
lrem key count（数量） value 根据元素的值删元素 可以指定删除数量

4.集合(set)
sadd key v1 v2 v3… 添加元素
srem key value 删除元素
smembers key 查看所有元素
sinter key1 key2 交集
sunion key1 key2 并集
sdiff key1 key2 差集 key1有 key2没有
scard key 查看元素个数
spop key 获取随机元素
sismember key value 查看该元素是否在该集合中

5.有序集合（zset）
zadd key score1 member1 score2 member2…
zrem key member
zrange key 0 -1 升序
zrevrange key 0 -1 降序
zscore key member 查看元素对应score
zincrby key score member
zdecrby key score member

LBS应用 - Locaton-based service -基于位置的服务

geoadd key location1 经度 纬度 location2 经度 纬度…
geodist key location1 location2
georadius key 经度 纬度 距离 withcoord（显示坐标） withdist（显示距离）
